This folder contains binary release files in the following folders:
  • compact-2.0
  • compact-3.5
  • net-2.0
  • net-3.5

The "compact" folders contain the following files:

Cecil.Flowanalysis.dll bytecode analyzing library for Native Query optimization
Db4oBench.dll IO benchmark library
Db4objects.Db4o.CS.dll db4o client/server class library
Db4objects.Db4o.dll db4o database engine
Db4objects.Db4o.Instrumentation.dll bytecode instrumentation library, required for enhancements for Native Queries and Transparent Activation
Db4objects.Db4o.Linq.dll Support for LINQ
Db4objects.Db4o.NativeQueries.dll Native Queries support library
Db4objects.Db4o.Optional.dll db4o optional components
Mono.Cecil.dll Mono Cecil bytecode analyzing library, required for enhancements for Native Queries and Transparent Activation
System.Linq.Expressions.dll Support for LINQ on CompactFramework


The "net" folders contain the following files in addition to the above:

Db4oTool.exe db4o commandline tool. For a list of options run it without parameters.
Db4oTool.MSBuild.dll MSBuild tasks for build-time enhancements for Transparent Activation and Native Queries
Mono.Getoptions.dll required for Db4oTool.exe to parse parameters
Mono.Cecil.Pdb.dll Mono Cecil pdb library
Moq.dll moq mocking library for .NET 3.5.
